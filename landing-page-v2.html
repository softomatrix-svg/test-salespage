The 6 Frameworks

Framework 1: The Role-Context-Task Architecture
This framework gets AI to adopt the right perspective before generating output. Instead of vague instructions, you create a complete context that guides the model's "thinking." The result: output that's immediately relevant and usable.

Framework 2: The Strategic Constraint Method
Paradoxically, adding limitations increases creativity and uniqueness. This framework uses carefully chosen constraints to force AI into generating distinctive, non-generic content that stands out.

Framework 3: The Chain-of-Thought Sequencing
Complex marketing tasks—like full email sequences or multi-part sales pages—break down into logical steps. This framework structures prompts so AI executes each step flawlessly, building toward a complete, cohesive result.

Framework 4: The Output Template Specification
Vague requests produce vague results. This framework ensures AI produces exactly the format you need—whether that's a specific email structure, social post format, or content outline.

Framework 5: The Iterative Refinement Protocol
Even good prompts sometimes need adjustment. This framework provides a systematic approach to turning good output into great output through targeted, efficient follow-up prompts.

Framework 6: The Multi-Persona Perspective Engine
Great marketing considers multiple viewpoints. This framework generates diverse perspectives and angles from a single prompt, giving you options and depth you wouldn't get otherwise.
