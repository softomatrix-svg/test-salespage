# Sales Page Template Library

A collection of proven, high-converting sales page templates ready to use.

## Available Templates

### 1. Professional Gold Template
**File:** `professional-gold-template.html`

**Best for:** Courses, coaching programs, professional services

**Style:**
- Clean, modern design
- White backgrounds with navy/dark sections
- Gold/yellow accents (#f0c040, #dfb83f)
- Georgia serif headings + Arial body text
- Large typography

**Sections:**
- Hero with dark navy (#15152b) background
- Video section (light gray)
- Problem section (white)
- Product intro (off-white)
- 6-component features grid
- 5-phase timeline
- 3-tier pricing
- Guarantee section
- FAQ
- Final CTA

**Live Example:** v3-bold deployment

## How to Use

1. Choose your template based on your product/style
2. Copy the HTML file
3. Replace all content with your own
4. Customize colors if needed
5. Deploy

## Naming Convention

Templates follow this naming pattern:
```
[STYLE]-[PRIMARY_COLOR]-template.html
```

Examples:
- `professional-gold-template.html`
- `dark-cyan-template.html`
- `minimal-black-template.html` (future)
- `bold-red-template.html` (future)

## Template Index

To recall a template, reference it by:
- **File name** (e.g., "professional-gold-template")
- **Style name** (e.g., "Professional Gold")
- **Best use case** (e.g., "course sales page")
