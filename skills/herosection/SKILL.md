# Hero Section Generator Skill

## Command: `/herosection`

Generate a 3-part headline structure for hero sections. Each part serves a different psychological function.

## Usage

Type `/herosection` and provide:
- Product/offer name
- Core mechanism
- Target audience
- Primary outcome

## Output

3-part headline structure:
1. **Pattern Interrupt** (3-7 words, bold, punchy)
2. **Mechanism Reveal** (clarity line explaining how)
3. **Outcome + Identity Shift** (what this means for them)

Plus supporting proof line, CTA block, and micro reassurance.
