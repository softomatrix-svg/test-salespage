# Business Builder AI V3

An OpenClaw skill for creating, validating, building, and launching a digital product or service business.

## Files

- `SKILL.md` — main skill file
- `README.md` — install and usage notes
- `config.example.json` — optional config template
- `guardrails.md` — safety and compliance checklist

## Install

1. Unzip the folder.
2. Copy `business-builder-ai-v3` into your OpenClaw skills folder.
3. Restart or reload OpenClaw.
4. Run:

```text
/create_app
```

## Important

Use Stripe test mode first. Do not paste API keys into public chats or logs.
