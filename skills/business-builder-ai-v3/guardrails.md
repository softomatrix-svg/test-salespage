# Safety and Guardrails Checklist

## Always Confirm Before

- Creating Stripe products
- Setting or changing prices
- Publishing a landing page
- Sending emails
- Running ads
- Making public claims

## Never Do

- Guarantee income
- Invent testimonials
- Use fake scarcity
- Expose API keys
- Store secrets in plain text
- Send spam emails

## Required Email Placeholders

- Unsubscribe link
- Business mailing address
- Sender identity

## Stripe Safety

- Use test mode first
- Confirm product name, description, price, currency, and billing type
- Confirm live mode separately

## Review Before Launch

- Pricing consistency
- Checkout link accuracy
- No false claims
- Clear refund policy
- Clear customer deliverables
