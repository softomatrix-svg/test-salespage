# AI PROMPT MASTERY - PROMPT LIBRARY (PART 2)
## 25 COPYWRITING PROMPTS

### Headlines & Hooks

**PROMPT 51: Headline Variations**
```
Write 10 headline variations for [PRODUCT/OFFER] using different angles: 1) Curiosity-driven, 2) Benefit-focused, 3) Problem-agitation, 4) How-to, 5) Numbered list, 6) Question-based, 7) News angle, 8) Contrarian, 9) Urgency/scarcity, 10) Social proof. Each should be under 15 words and make the reader want to learn more.
```

**PROMPT 52: Hook Writing**
```
Write 5 opening hooks for [SALES PAGE/EMAIL/AD] that: 1) Start with a bold statement, 2) Ask a thought-provoking question, 3) Share a surprising statistic, 4) Tell a micro-story (2-3 sentences), 5) Use the "Imagine..." framework. Each hook should stop the scroll and create curiosity about what comes next.
```

**PROMPT 53: Subheadline Creation**
```
Write 3 subheadlines that support this headline: "[HEADLINE]". Each subheadline should: 1) Expand on the promise, 2) Add specificity or proof, 3) Bridge to the body copy. Make them feel like natural extensions of the main headline, not repetitive.
```

**PROMPT 54: Button Copy**
```
Write 10 CTA button variations for [OFFER] that aren't generic "Buy Now" or "Click Here". Use: 1) Action + benefit, 2) First-person, 3) Urgency without being pushy, 4) Outcome-focused, 5) Curiosity-driven. Keep to 2-4 words each. Make them feel like the obvious next step.
```

**PROMPT 55: Bullet Point Benefits**
```
Transform these features [LIST FEATURES] into benefit-driven bullet points using the "So that..." framework. Format: Feature + "which means" + Benefit + "so that" + Ultimate outcome. Write 10 bullets that make the reader feel the transformation.
```

### Sales Pages

**PROMPT 56: Problem-Agitation-Solution**
```
Write a PAS section for [PRODUCT] targeting [AUDIENCE]. Problem: Describe their pain in vivid detail (2-3 sentences). Agitation: Make it feel urgent and emotional—what happens if this continues? (3-4 sentences). Solution: Introduce your product as the answer, focusing on the transformation, not features (2-3 sentences).
```

**PROMPT 57: Unique Mechanism**
```
Write a "unique mechanism" section for [PRODUCT] that explains: 1) Why other solutions fail (the old way), 2) The breakthrough discovery or method you use (the new way), 3) How it works in simple terms, 4) Why it's different/better. Make it feel like a genuine innovation, not marketing fluff.
```

**PROMPT 58: Objection Handling**
```
Write responses to these 5 objections for [PRODUCT]: 1) Price too high, 2) Not the right time, 3) Need to think about it, 4) Tried similar before and it didn't work, 5) Need to ask spouse/boss. For each: acknowledge the concern, reframe it, provide proof/counter-example, and transition back to the offer.
```

**PROMPT 59: Guarantee Section**
```
Write a risk-reversal guarantee for [PRODUCT] that: 1) States the guarantee clearly (timeframe + terms), 2) Addresses the "catch" objection upfront, 3) Explains why you can offer this (confidence in product), 4) Makes it easy to claim, 5) Ends with confidence. Make it feel generous but genuine.
```

**PROMPT 60: Urgency/Scarcity**
```
Write 3 variations of urgency for [OFFER] that feel real, not fake: 1) Time-based (deadline), 2) Quantity-based (limited spots), 3) Price-based (increasing soon). Each should explain WHY the urgency exists (not just "act now"). Make it believable and ethical.
```

### Ads

**PROMPT 61: Facebook Ad Primary Text**
```
Write Facebook ad copy (primary text) for [PRODUCT] targeting [AUDIENCE]. Include: 1) Pattern interrupt hook (first 2 lines), 2) Problem identification, 3) Solution tease, 4) Social proof element, 5) Soft CTA. Keep under 125 characters for the "see more" cutoff. Make it feel native to Facebook, not salesy.
```

**PROMPT 62: Facebook Ad Headlines**
```
Write 5 Facebook ad headlines (under 40 characters each) for [OFFER]: 1) Curiosity-driven, 2) Benefit-focused, 3) Question-based, 4) How-to format, 5) Social proof angle. Each should make people stop scrolling and want to read more.
```

**PROMPT 63: Google Search Ad**
```
Write a Google Search ad for [KEYWORD] with: Headline 1 (30 chars): Include keyword + benefit. Headline 2 (30 chars): Feature or proof. Headline 3 (30 chars): CTA. Description 1 (90 chars): Expand on value proposition. Description 2 (90 chars): Include social proof or urgency. Make it match search intent perfectly.
```

**PROMPT 64: YouTube Pre-Roll Script**
```
Write a 30-second YouTube pre-roll ad script for [PRODUCT]. Structure: 0-5s: Hook that stops the "skip", 5-15s: Problem identification, 15-25s: Solution + proof, 25-30s: CTA. Write for speaking, not reading. Include [VISUAL] notes. Make it punchy and impossible to ignore.
```

**PROMPT 65: Retargeting Ad**
```
Write retargeting ad copy for people who visited [LANDING PAGE] but didn't buy. Acknowledge they've seen the offer (without being creepy), address the #1 objection they likely have, add new information or social proof they haven't seen, and create gentle urgency. Tone: understanding, not pushy.
```

### Landing Pages

**PROMPT 66: Above-the-Fold Section**
```
Write the above-the-fold section for [LANDING PAGE]. Include: 1) Headline that speaks to the #1 desire or pain, 2) Subheadline that adds specificity or proof, 3) 3 bullet benefits, 4) CTA button, 5) Trust element (social proof or guarantee). Everything should fit without scrolling. Make the CTA feel like the obvious next step.
```

**PROMPT 67: Features vs Benefits**
```
Take these features [LIST FEATURES] and rewrite them as benefits using this formula: "[Feature], which means [Direct Benefit], so you can [Ultimate Outcome]." Create 8-10 benefit statements that make the reader envision their life after purchasing.
```

**PROMPT 68: FAQ Section**
```
Write 8 FAQ items for [PRODUCT] covering: 1) How it works, 2) Who it's for/not for, 3) Time to results, 4) Comparison to alternatives, 5) Support included, 6) Refund policy, 7) Technical requirements, 8) Getting started. Write answers that overcome objections while being genuinely helpful.
```

**PROMPT 69: Testimonial Request**
```
Write an email requesting testimonials from customers who've used [PRODUCT]. Ask specific questions that will generate detailed, results-focused testimonials: 1) What was the problem before? 2) What made you try this? 3) What specific results did you get? 4) What would you tell someone considering this? Make it easy for them to respond.
```

**PROMPT 70: Pricing Page**
```
Write a pricing page for [PRODUCT] with 3 tiers: Basic ($[PRICE]), Pro ($[PRICE]), Enterprise ($[PRICE]). For each: Name, Price, 4-5 features, "Best for" description, CTA. Include a "Most Popular" highlight on the middle tier. Add a comparison table showing what's included in each. Make the middle tier feel like the obvious choice.
```

### Email Sequences

**PROMPT 71: Launch Sequence (5 Emails)**
```
Outline a 5-email launch sequence for [PRODUCT]: Email 1 - Tease/coming soon, Email 2 - Problem agitation, Email 3 - Solution reveal + early bird, Email 4 - Social proof + case studies, Email 5 - Cart open + urgency. Include subject lines, key points for each, and overall narrative arc.
```

**PROMPT 72: Post-Purchase Sequence**
```
Write a 3-email post-purchase sequence: Email 1 - Welcome + quick win (deliver immediate value), Email 2 - Success story + advanced tip (show what's possible), Email 3 - Upsell/cross-sell (next logical step). Make each email feel like a gift, not a sales pitch.
```

**PROMPT 73: Re-engagement Campaign**
```
Write a 3-email re-engagement sequence for inactive subscribers: Email 1 - "We miss you" + what's new, Email 2 - Best content they missed + value reminder, Email 3 - Last chance + unsubscribe option (with feedback request). Be warm, not desperate. Give them an easy out.
```

**PROMPT 74: Webinar Follow-Up**
```
Write a 3-email webinar follow-up sequence: Email 1 (same day) - Replay + key takeaways, Email 2 (2 days later) - Address objections + FAQ, Email 3 (4 days later) - Final chance + urgency. Reference specific moments from the webinar to build connection.
```

**PROMPT 75: Affiliate Promotion**
```
Write an email promoting [AFFILIATE PRODUCT] to your list. Include: 1) Personal story of why you use/love it, 2) Who it's perfect for (and who should skip it), 3) Specific results you've seen, 4) What's included, 5) Bonus you're offering, 6) CTA with deadline. Be honest about the affiliate relationship.
```

## 25 CONTENT STRATEGY PROMPTS

### Blog Posts

**PROMPT 76: Blog Post Outline**
```
Create a detailed outline for a blog post about [TOPIC] targeting [AUDIENCE]. Include: 1) Working title (SEO-optimized), 2) Target keyword, 3) Hook/introduction approach, 4) 5-7 main sections with sub-points, 5) Key takeaways, 6) CTA. Make it comprehensive enough to be the definitive guide on this topic.
```

**PROMPT 77: List Post**
```
Write a list post: "[NUMBER] Ways to [ACHIEVE RESULT]". Include: 1) Hook that promises value, 2) Brief intro, 3) [NUMBER] items with detailed explanations (200+ words each), 4) Actionable takeaways for each, 5) Conclusion with CTA. Make each item genuinely useful, not filler.
```

**PROMPT 78: How-To Guide**
```
Write a comprehensive how-to guide for [PROCESS/TASK]. Structure: 1) What the reader will learn, 2) Why it matters, 3) Prerequisites/tools needed, 4) Step-by-step instructions (with screenshots noted), 5) Common mistakes to avoid, 6) Advanced tips, 7) Next steps. Make it actionable enough to follow along.
```

**PROMPT 79: Case Study**
```
Write a case study about [CUSTOMER/PROJECT]. Include: 1) Background on the subject, 2) The challenge/problem (with specifics), 3) The solution/approach taken, 4) Implementation details, 5) Results (with numbers/metrics), 6) Key lessons, 7) How reader can apply this. Make it story-driven but data-backed.
```

**PROMPT 80: Comparison Post**
```
Write a comparison post: "[OPTION A] vs [OPTION B]: Which is Right for You?" Include: 1) Brief overview of both, 2) 5-7 comparison categories, 3) Pros/cons of each, 4) Who each is best for, 5) Clear recommendation with reasoning, 6) CTA. Be fair but opinionated. Help readers make a decision.
```
