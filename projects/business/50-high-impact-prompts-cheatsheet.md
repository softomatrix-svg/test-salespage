## EMAIL MARKETING (8 PROMPTS)

### 1. Welcome Email #1 - First Impression
```
You are a conversion copywriter with 10 years of experience writing welcome emails for [INDUSTRY]. Write the first email in a welcome sequence for [PRODUCT/SERVICE]. This email should: 1) Thank them for joining, 2) Set expectations for what's coming, 3) Deliver immediate value with [SPECIFIC TIP/RESOURCE], 4) Include a soft CTA to [ACTION]. Tone: [FRIENDLY/PROFESSIONAL]. Length: 150-200 words.
```
**Use for:** Any welcome sequence, onboarding, new subscriber emails

### 2. Sales Email - Direct Offer
```
Write a sales email for [PRODUCT NAME]. Include: 1) Attention-grabbing subject line, 2) Opening that identifies the problem ([PAIN POINT]), 3) The solution and how it works, 4) 3 key benefits with specifics, 5) Social proof (testimonial or case study), 6) Clear CTA with urgency. Tone: enthusiastic but professional. Length: 250-300 words.
```
**Use for:** Product launches, flash sales, special offers

### 3. Abandoned Cart Recovery
```
Write an abandoned cart email for [PRODUCT]. Include: 1) Friendly reminder they left something behind, 2) Brief description of what they're missing, 3) One key benefit they haven't experienced yet, 4) Social proof (reviews or testimonials), 5) Clear CTA to complete purchase, 6) Optional: small discount or free shipping offer. Tone: helpful, not pushy. Length: 150-200 words.
```
**Use for:** E-commerce, course sales, any checkout abandonment

### 4. Testimonial Feature Email
```
Write an email featuring a customer success story: 1) Subject line that creates curiosity, 2) Introduce [CUSTOMER NAME] and their situation, 3) The problem they faced (be specific), 4) How they used [PRODUCT/SERVICE] to solve it, 5) The specific results (numbers), 6) CTA to achieve similar results. Use quotes from the customer. Tone: story-driven and inspiring. Length: 250-300 words.
```
**Use for:** Social proof emails, case study features, trust building

### 5. Re-engagement - Win Back
```
Write a win-back email for subscribers who haven't opened emails in [TIME PERIOD]. Include: 1) Subject line that acknowledges the silence, 2) "We miss you" opening, 3) Quick recap of what they've missed (value), 4) One piece of recent content they should see, 5) Easy way to update preferences, 6) Soft CTA to re-engage. Don't guilt-trip. Tone: warm and understanding. Length: 200-250 words.
```
**Use for:** Inactive subscribers, list cleaning, re-activation campaigns

### 6. Educational Value Email
```
Write an educational email teaching [SPECIFIC TOPIC/CONCEPT]. Include: 1) Hook that identifies a common mistake, 2) Explain the concept in simple terms, 3) Why most people get it wrong, 4) The right way to do it (step-by-step), 5) Real example or case study, 6) Soft CTA to learn more. Make it genuinely helpful with actionable takeaways. Tone: teacher/expert. Length: 300-400 words.
```
**Use for:** Nurture sequences, authority building, value-first marketing

### 7. FAQ Email - Objection Handling
```
Write an FAQ email addressing the top 5 questions about [PRODUCT/TOPIC]. Format: 1) Brief intro, 2) Q1 with detailed A1, 3) Q2 with detailed A2, 4) Q3 with detailed A3, 5) Q4 with detailed A4, 6) Q5 with detailed A5, 7) CTA to ask more questions. Make answers thorough but scannable. Tone: helpful and patient. Length: 400-500 words.
```
**Use for:** Pre-launch, sales pages, customer support prevention

### 8. Last Chance Urgency Email
```
Write a "last chance" email for [OFFER/PRODUCT]. Include: 1) Subject line with urgency, 2) Reminder of what they're losing access to, 3) Recap of key benefits, 4) Social proof (how many people have already taken action), 5) Clear deadline (specific date/time), 6) Strong final CTA. Make it feel like a genuine last opportunity. Tone: urgent but respectful. Length: 200-250 words.
```
**Use for:** Cart close, offer expiration, final reminders

## SOCIAL MEDIA (8 PROMPTS)

### 9. LinkedIn Story Post - Professional Narrative
```
Write a LinkedIn post telling a professional story about [EXPERIENCE/CHALLENGE/SUCCESS]. Structure: 1) Hook in first 2 lines, 2) Setup the situation, 3) The conflict or challenge, 4) The turning point, 5) The resolution and lesson, 6) CTA for engagement. Use line breaks for readability. Include 3 relevant hashtags. Tone: authentic and professional. Length: 150-200 words.
```
**Use for:** Personal branding, thought leadership, engagement building

### 10. Twitter Thread - How-To Tutorial
```
Write a Twitter/X thread teaching [SPECIFIC SKILL/CONCEPT] in 8-10 tweets. Structure: 1) Hook tweet (problem + promise), 2) Setup/context, 3-7) Step-by-step breakdown (one step per tweet), 8) Common mistakes, 9) Advanced tip, 10) CTA + link. Each tweet should stand alone but flow together. Use formatting (numbers, emojis, line breaks). Make it bookmark-worthy.
```
**Use for:** Educational content, authority building, viral potential

### 11. Instagram Carousel - Educational
```
Write an Instagram carousel post about [TOPIC]. Create 5-7 slides: 1) Hook slide (the problem), 2) Context slide, 3) Key insight #1, 4) Key insight #2, 5) Key insight #3, 6) Example/case study, 7) CTA slide. Each slide should be 1-2 sentences max. Make it swipe-worthy. Include hashtag strategy. Tone: educational and valuable.
```
**Use for:** Instagram growth, saves/shares, educational content

### 12. Facebook Long-Form Post
```
Write a Facebook post (long-form) about [TOPIC/STORY]. Include: 1) Hook in first 2 lines, 2) Personal story or experience, 3) The lesson or insight, 4) How it applies to the reader, 5) Actionable advice, 6) CTA for comments. Facebook favors longer engagement, so make it meaty. Length: 300-500 words.
```
**Use for:** Facebook engagement, community building, storytelling

### 13. Social Proof Post
```
Write a social media post featuring customer results/testimonials. Include: 1) Hook about the transformation, 2) Before state (the problem), 3) The solution they used, 4) After state (specific results with numbers), 5) Their quote/testimonial, 6) CTA for similar results. Make the customer the hero. Include permission note.
```
**Use for:** Trust building, conversion boosting, testimonial showcases

### 14. Contrarian/Hot Take Post
```
Write a contrarian LinkedIn post challenging conventional wisdom about [TOPIC]. Structure: 1) Popular belief everyone accepts, 2) Why it's wrong or incomplete, 3) The alternative view, 4) Proof or example, 5) Call to question assumptions. Be respectful but provocative. Include 3 hashtags. Tone: thought leadership. Length: 150-200 words.
```
**Use for:** Thought leadership, engagement spikes, standing out

### 15. List Post - High Value
```
Write a LinkedIn list post: "[NUMBER] [TOPIC] mistakes I see [AUDIENCE] make". Include: 1) Hook in first line, 2) Brief intro, 3) [NUMBER] mistakes with brief explanations, 4) How to avoid them, 5) CTA. Make each point actionable. Use emojis for visual breaks. Include 3 hashtags. Tone: helpful and authoritative. Length: 200-250 words.
```
**Use for:** Listicles, educational content, high engagement

### 16. Engagement Question Post
```
Write a social media post designed to maximize engagement. Ask a thought-provoking question about [TOPIC] that: 1) Relates to a common pain point, 2) Has no obvious answer, 3) Invites different perspectives, 4) You can respond to comments on. Keep it short (50-100 words) with a clear question at the end. Include 3 hashtags. Tone: curious and inclusive.
```
**Use for:** Community building, algorithm boost, market research

## COPYWRITING (9 PROMPTS)

### 17. Headline Variations - Test Different Angles
```
Write 10 headline variations for [PRODUCT/OFFER] using different angles: 1) Curiosity-driven, 2) Benefit-focused, 3) Problem-agitation, 4) How-to, 5) Numbered list, 6) Question-based, 7) News angle, 8) Contrarian, 9) Urgency/scarcity, 10) Social proof. Each should be under 15 words and make the reader want to learn more.
```
**Use for:** A/B testing, finding winning headlines, sales pages

### 18. Problem-Agitation-Solution (PAS)
```
Write a PAS section for [PRODUCT] targeting [AUDIENCE]. Problem: Describe their pain in vivid detail (2-3 sentences). Agitation: Make it feel urgent and emotional—what happens if this continues? (3-4 sentences). Solution: Introduce your product as the answer, focusing on the transformation, not features (2-3 sentences).
```
**Use for:** Sales pages, landing pages, conversion optimization

### 19. Bullet Point Benefits - Feature Translation
```
Transform these features [LIST FEATURES] into benefit-driven bullet points using the "So that..." framework. Format: Feature + "which means" + Benefit + "so that" + Ultimate outcome. Write 10 bullets that make the reader feel the transformation.
```
**Use for:** Sales pages, product descriptions, email copy

### 20. Facebook Ad Primary Text
```
Write Facebook ad copy (primary text) for [PRODUCT] targeting [AUDIENCE]. Include: 1) Pattern interrupt hook (first 2 lines), 2) Problem identification, 3) Solution tease, 4) Social proof element, 5) Soft CTA. Keep under 125 characters for the "see more" cutoff. Make it feel native to Facebook, not salesy.
```
**Use for:** Facebook/Instagram ads, paid social campaigns

### 21. Google Search Ad
```
Write a Google Search ad for [KEYWORD] with: Headline 1 (30 chars): Include keyword + benefit. Headline 2 (30 chars): Feature or proof. Headline 3 (30 chars): CTA. Description 1 (90 chars): Expand on value proposition. Description 2 (90 chars): Include social proof or urgency. Make it match search intent perfectly.
```
**Use for:** Google Ads, search campaigns, high-intent traffic

### 22. Landing Page Above-the-Fold
```
Write the above-the-fold section for [LANDING PAGE]. Include: 1) Headline that speaks to the #1 desire or pain, 2) Subheadline that adds specificity or proof, 3) 3 bullet benefits, 4) CTA button, 5) Trust element (social proof or guarantee). Everything should fit without scrolling. Make the CTA feel like the obvious next step.
```
**Use for:** Landing pages, squeeze pages, conversion optimization

### 23. Objection Handling - FAQ Style
```
Write responses to these 5 objections for [PRODUCT]: 1) Price too high, 2) Not the right time, 3) Need to think about it, 4) Tried similar before and it didn't work, 5) Need to ask spouse/boss. For each: acknowledge the concern, reframe it, provide proof/counter-example, and transition back to the offer.
```
**Use for:** Sales pages, overcoming objections, closing sales

### 24. Button Copy - CTA Optimization
```
Write 10 CTA button variations for [OFFER] that aren't generic "Buy Now" or "Click Here". Use: 1) Action + benefit, 2) First-person, 3) Urgency without being pushy, 4) Outcome-focused, 5) Curiosity-driven. Keep to 2-4 words each. Make them feel like the obvious next step.
```
**Use for:** CTA buttons, conversion optimization, A/B testing

### 25. Guarantee/Risk Reversal
```
Write a risk-reversal guarantee for [PRODUCT] that: 1) States the guarantee clearly (timeframe + terms), 2) Addresses the "catch" objection upfront, 3) Explains why you can offer this (confidence in product), 4) Makes it easy to claim, 5) Ends with confidence. Make it feel generous but genuine.
```
**Use for:** Risk removal, trust building, increasing conversions

Make it story-driven but data-backed.

### 29. Lead Magnet - Checklist

### 30. Video Script - YouTube

Include [B-ROLL] and [GRAPHIC] notes. Write for speaking, not reading.

### 31. Content Repurposing Plan

**Use for:** Content efficiency, maximizing reach, multi-platform strategy

### 32. Content Calendar - 30 Days

### 33. Webinar Outline - Sales Presentation


## SEO (9 PROMPTS)

### 34. Keyword Brainstorming
```
Generate 50 keyword ideas for [TOPIC/INDUSTRY] across different intents: 1) Informational (how-to, what is, guide), 2) Navigational (brand + topic), 3) Commercial (best, top, vs, review), 4) Transactional (buy, discount, free shipping). Include long-tail variations (4+ words). Make them specific enough to have search volume but not so competitive they're impossible to rank for.
```
**Use for:** Keyword research, content planning, SEO strategy

### 35. Title Tag Optimization
```
Write 5 optimized title tags for [PAGE] targeting [KEYWORD]. Each should: 1) Include the exact keyword near the beginning, 2) Stay under 60 characters, 3) Include a benefit or hook, 4) Be click-worthy. Follow format: Primary Keyword - Benefit/Value Proposition | Brand
```
**Use for:** On-page SEO, click-through rate optimization

### 36. Meta Description Writing
```
Write 3 meta descriptions for [PAGE] targeting [KEYWORD]. Each should: 1) Include the keyword naturally, 2) Stay under 155 characters, 3) Include a clear benefit, 4) Have a soft CTA, 5) Match search intent. Make them compelling enough to earn the click from SERP.
```
**Use for:** SERP optimization, click-through rate improvement

### 37. Content Brief Creation
```
Create a comprehensive content brief for a blog post targeting [KEYWORD]. Include: 1) Target keyword + 5 secondary keywords, 2) Search intent analysis, 3) Target word count, 4) Required sections/outline, 5) Questions to answer, 6) Internal/external link targets, 7) Content format (listicle, guide, etc.), 8) Call-to-action. Make it detailed enough for any writer to use.
```
**Use for:** Content creation, team coordination, SEO briefs

### 38. Featured Snippet Optimization
```
Optimize this content [PASTE CONTENT] for featured snippets. Identify: 1) Paragraph snippet opportunities (40-60 words), 2) List snippet opportunities (numbered/bulleted), 3) Table snippet opportunities, 4) Question-based headings. Rewrite sections to increase snippet chances while maintaining quality.
```
**Use for:** Position zero rankings, featured snippets, voice search

### 39. Content Refresh Strategy

```
Analyze this old blog post [PASTE URL/CONTENT] and create a refresh plan: 1) Outdated information to update, 2) New sections to add, 3) Keywords to add/remove, 4) Internal links to add, 5) Content to remove/condense, 6) New title/meta description. Show how to improve rankings without starting from scratch.
```

**Use for:** Content updates, improving old posts, SEO maintenance

### 40. Local SEO - Google Business Profile

```
Write optimized content for a Google Business Profile for [BUSINESS NAME] in [INDUSTRY/LOCATION]. Include: 1) Business description (750 characters), 2) 5 service descriptions, 3) Q&A section (5 common questions + answers), 4) Post ideas for the next 30 days. Make it keyword-rich but natural.
```

### 41. SEO Audit Checklist

```
Create a comprehensive SEO audit checklist covering: 1) Technical SEO (20 points), 2) On-page SEO (20 points), 3) Content (15 points), 4) Off-page/Links (15 points), 5) Local SEO (10 points), 6) Analytics/Tracking (10 points). Each item should be actionable and include how to check it.
```

## ADVERTISING (8 PROMPTS)

### 43. Facebook Ad Campaign Structure
```
Design a Facebook ad campaign structure for [PRODUCT/OFFER] with: 1) Campaign objective, 2) 3 ad sets (audiences), 3) 2-3 ads per ad set (creative variations), 4) Budget allocation, 5) Testing strategy. Include targeting suggestions and creative angles for each audience.
```
**Use for:** Facebook/Instagram ad campaigns, paid social strategy

### 44. Google Search Campaign
```
Design a Google Search campaign for [PRODUCT/SERVICE] with: 1) Campaign structure (3-5 ad groups), 2) Keyword match types strategy, 3) Negative keyword suggestions, 4) Ad copy variations (3 per ad group), 5) Landing page recommendations, 6) Budget and bidding strategy. Focus on high-intent keywords.
```
**Use for:** Google Ads, search campaigns, PPC strategy

### 45. Retargeting Ad Sequence
```
Create a 3-ad retargeting sequence for cart abandoners: Ad 1 (Day 1): Gentle reminder, Ad 2 (Day 3): Social proof + objection handling, Ad 3 (Day 5): Final offer + urgency. Each ad should acknowledge where they are in the journey without being creepy.
```
**Use for:** Cart recovery, abandoned checkout, conversion optimization

### 46. YouTube Pre-Roll Script
```
Write a 30-second YouTube pre-roll ad script for [PRODUCT]: 0-5s: Hook that stops the skip, 5-15s: Problem identification, 15-25s: Solution + proof, 25-30s: CTA. Write for speaking, not reading. Include [VISUAL] notes and make it punchy. The first 5 seconds are everything.
```
**Use for:** YouTube ads, video marketing, brand awareness

### 47. Ad Creative Concepts
```
Generate 10 ad creative concepts for [PRODUCT]: 1) Problem-solution format, 2) Before/after transformation, 3) Testimonial/social proof, 4) Educational/tip-based, 5) Story-driven, 6) Urgency/scarcity, 7) Question-based, 8) List/bullet points, 9) Video hook ideas, 10) Carousel concepts. Include visual descriptions.
```
**Use for:** Creative testing, ad variations, campaign planning

### 48. Landing Page for Ads
```
Write a dedicated landing page for [AD CAMPAIGN] that: 1) Matches ad messaging exactly, 2) Has one clear CTA, 3) Loads fast (minimal scripts), 4) Has social proof above the fold, 5) Addresses main objections, 6) Has no navigation distractions. Focus on conversion, not information.
```
**Use for:** Landing pages, paid traffic, conversion optimization

### 49. Customer Avatar for Ads
```
Create a detailed customer avatar for [PRODUCT] ad targeting: 1) Demographics (age, location, income), 2) Psychographics (values, interests, fears), 3) Pain points (3-5 specific), 4) Desired outcomes, 5) Objections, 6) Where they hang out online. Make it detailed enough to write targeted ad copy.
```
**Use for:** Audience targeting, ad copywriting, campaign planning

### 50. Ad Testing Framework
```
Create a systematic ad testing plan for [CAMPAIGN]: 1) What to test first (creative, copy, audience, offer), 2) Testing methodology (A/B vs multivariate), 3) Success metrics and minimum sample sizes, 4) Testing timeline, 5) How to implement winners. Make it scientific and scalable.
```
**Use for:** Ad optimization, testing strategy, performance improvement
